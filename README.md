# aha-clone-ott-platform

## 🚀 Description

A responsive OTT platform clone inspired by Aha. The application provides a modern UI with media content display, authentication, and interactive carousels.

## 🛠️ Tech Stack

* Frontend: HTML, CSS, Bootstrap, JavaScript
* Tools: Firebase (for hosting/authentication)

## ✨ Features

* Responsive UI design
* User authentication
* Media carousel display
* Clean and modern interface

## ▶️ How to Run

1. Open project in browser
2. (Optional) Deploy using Firebase
3. Explore UI and features
